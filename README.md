# blazeQOL

## Setup

Use command `/display on` to enable the features or `/display off` to disable them.

## Features

A HUD in the middle of the screen showing you name, health and attunement of the Blaze Slayer boss with the indicator whether
Twilight Arrow Poison has been used in a range of 8 blocks from the player.

minecraft version 1.21.11
